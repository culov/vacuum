# Vacuum

[![travis][1]][2]

Vacuum is a thin Ruby wrapper to the [Amazon Product Advertising API][3].

![vacuum][4]

```ruby
request = Vacuum.new

request.configure key:    'foo',
                  secret: 'secret',
                  tag:    'foo-bar'

request.get query: { 'Operation'   => 'ItemSearch',
                     'SearchIndex' => 'Books',
                     'Keywords'    => 'Architecture' }
```
